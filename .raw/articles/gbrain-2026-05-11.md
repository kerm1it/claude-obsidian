# GBrain — AI Agent Memory & Knowledge Management System

Author: Garry Tan (President & CEO, Y Combinator)
Stars: 14.6K | Forks: 1.9K | License: MIT | Language: TypeScript (98%)

## Overview

GBrain is an AI agent memory and knowledge management system — "a brain" for AI agents that are "smart but forgetful." It powers Tan's actual OpenClaw and Hermes deployments, managing 17,888 pages, 4,383 people, and 723 companies with 21 autonomous cron jobs.

## Core Capabilities

### Knowledge Management
- Hybrid search combining vector embeddings, keyword matching, and reciprocal rank fusion (RRF)
- Self-wiring knowledge graph with typed relationships (attended, works_at, invested_in, founded, advises)
- Compiled truth + timeline pattern for all pages
- Automatic entity extraction and backlink generation

### Agent Integration
- 34 specialized skills for content ingestion, enrichment, and operations
- Minions job queue for deterministic background work
- MCP server support (Claude, Cursor, Windsurf, remote OAuth)
- Durable subagent execution with crash recovery

### Data Ingestion
- Meetings, emails, tweets, voice calls, articles, PDFs
- Integration recipes for Gmail, Calendar, X/Twitter, Twilio voice
- Automatic entity enrichment and cross-referencing

## Key Concepts

### Compiled Truth + Timeline Pattern
All pages split into: compiled truth (stable facts, last-write-wins) + timeline (append-only events). Agents read the compiled section for fast context, and append to timeline for new events.

### Skillify
Converts one-off agent fixes into durable, tested, routed skills with 10-item conformance checklist. "Fat markdown documents encoding workflows."

### Dream Cycle
Automated overnight maintenance: synthesis, pattern detection, backlink enforcement, citation fixing.

### Minions Job Queue
Deterministic background work queue. Handles bulk enrichment, cross-referencing, and maintenance tasks without blocking agent sessions.

### Storage Tiering
Separates git-tracked (people, companies, key entities) from database-only (bulk content) directories.
