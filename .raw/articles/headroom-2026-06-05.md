# Headroom

headroom 是一个本地运行的 AI Agent 上下文压缩层。在工具输出、日志、RAG 结果、文件和对话历史到达 LLM 之前进行压缩——60-95% token 节省，答案不变。

## 架构

```
Agent/App → CacheAligner → ContentRouter → CCR
                           ├─ SmartCrusher (JSON)
                           ├─ CodeCompressor (AST)
                           └─ Kompress-base (text, HF model)
         → LLM Provider
```

- ContentRouter: 检测内容类型，选择正确压缩器
- SmartCrusher: 通用 JSON 压缩
- CodeCompressor: AST 感知（Python/JS/Go/Rust/Java/C++）
- Kompress-base: HuggingFace 模型，基于 agentic traces 训练
- CacheAligner: 稳定前缀 → KV cache 命中
- CCR: 可逆压缩，原文本地保存，LLM 按需取回

## 六种接入方式

1. Library: compress(messages) in Python/TypeScript
2. Proxy: headroom proxy --port 8787
3. Agent wrap: headroom wrap claude|codex|cursor|aider|copilot
4. MCP server: headroom_compress / headroom_retrieve / headroom_stats
5. Cross-agent memory: 共享存储，跨 Claude/Codex/Gemini
6. headroom learn: 挖掘失败 session → 写回 CLAUDE.md/AGENTS.md
