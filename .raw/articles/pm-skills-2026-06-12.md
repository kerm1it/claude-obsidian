# PM Skills Marketplace — The AI Operating System for Better Product Decisions

**作者**: phuryn
**GitHub**: https://github.com/phuryn/pm-skills
**最新版本**: v2.0.0 (2026-06-05)
**许可**: MIT | ⭐ 16K | 60 commits

## 概述

PM Skills Marketplace 是一个面向产品管理的 Agent Skills 市场，包含 68 个 skills、42 个 commands，分布在 9 个可安装 plugin 中，覆盖从发现到策略、执行、发布和增长的完整 PM 工作流。

**核心主张**："Generic AI gives you text. PM Skills Marketplace gives you structure."——每个 skill 将已建立的 PM 框架（Teresa Torres、Marty Cagan、Alberto Savoia 等）编码为有引导的逐步工作流。

## 架构三层

- **Skills**：原子构建块，提供领域知识、分析框架或引导工作流。当与对话上下文相关时自动加载
- **Commands**：用户触发的斜杠命令工作流（`/command-name`），将多个 skills 串联为端到端流程。完成后建议关联命令
- **Plugins**：将相关 skills 和 commands 分组成 9 个领域的可安装包

## 安装方式

Claude Cowork：GUI Marketplace 搜索 `phuryn/pm-skills`
Claude Code：`claude plugin marketplace add phuryn/pm-skills`
Codex CLI：`codex plugin marketplace add phuryn/pm-skills`
其他 Agent：Skills 文件（`skills/*/SKILL.md`）兼容 Gemini CLI、OpenCode、Cursor、Kiro

## 配套项目

[[entities/PMBrain]]（`phuryn/pm-brain`）：PM 的"第二大脑"，纯 Markdown 文件，无向量库或云依赖。

## 关键设计

- Commands 完成后建议相关下一步命令，匹配自然 PM 工作流
- 所有 skills 使用通用 Agent Skills 格式（SKILL.md），跨平台兼容
- pm-ai-shipping 插件解决"AI agent 写代码快但不留意图记录"的问题——文档化系统并审计文档与实际代码的差距
