# Agent 个人项目

个人 Agent 相关的动手实践项目。

---

## 待办

### 1. 金融持仓分析 Agent

- **目标**：创建一个个人金融 Agent，能分析持仓、给出意见和建议
- **范围**：
  - 读取持仓数据（可能从券商 API 或手动导入）
  - 分析持仓结构（行业分布、风险暴露、集中度）
  - 给出参考意见和建议
- **参考**：
  - [[concepts/Skillify]] — 可以从一次性的分析脚本固化为可复用 skill
  - [[entities/Ponytail]] — Agent 行为工具包，可用于控制 agent 输出质量
- **状态**：📋 待开始

### 2. 个人 Agent Plugin（Skill 集合）

- **目标**：创建自己的 Agent plugin，包含自己创建的 skill 集合
- **范围**：
  - 遵循 Agent Skills Standard（SKILL.md 格式）
  - 可以参考 [[entities/PMSkills]] 的 Skills/Commands/Plugins 三层架构
  - 可包含的 skill 候选项：金融分析、笔记处理、代码审查偏好等
- **参考**：
  - [[concepts/技能市场]] — Skills→Commands→Plugins 三层架构
  - [[concepts/AgentSkillsStandard]] — Agent Skills 规范
  - [[entities/PMSkills]] — PM 领域的 skill marketplace 实现案例
  - [[entities/ScientificAgentSkills]] — 科研领域的 skill repository 案例
- **状态**：📋 待开始
