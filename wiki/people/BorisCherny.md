## 关键观点

- **编程已被解决**：自 2025 年 10/11 月起，100% 的代码由模型编写；单日最高 150 个 PR
- **产品过剩**（[[concepts/ProductOverhang]]）：在模型就绪前就开始构建，等待模型赶上
- **Loop 是未来**（[[concepts/Loop机制]]）：cron 调度的重复 Agent 任务；几百个 Agent 并行，每晚几千个
- **编程民主化**（[[concepts/软件民主化]]）：类比印刷机，编程将成为大众技能
- 主要从**手机**工作：Claude App → Code 标签页
