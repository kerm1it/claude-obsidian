**职位：** Claude Code & Cowie 工程与产品负责人，[[entities/Anthropic]]

## 关于

Fiona Fung 领导 Anthropic 的 Claude Code 和 Cowie（Claude.ai）工程与产品团队，与 Boris Cherny 和 Kat 紧密合作。她在 2026 年的 Code with Claude 大会上分享了 AI 原生工程组织的实践经验。

## 核心观点

- **瓶颈已转移**：编码不再是瓶颈，验证/安全/跨职能协作才是
- **管理者先做 IC**：Claude Code 团队要求每位新入职管理者先以 IC 身份工作，建立「狗粮」能力和团队信任
- **代码赢技术争论**：技术分歧时直接生成多个 PR 对比，比白板讨论更高效
- **明确许可干掉旧流程**：流程不会自我消亡，需要团队领导主动授权清除
