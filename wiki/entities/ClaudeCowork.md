# Claude Cowork

Anthropic 推出的 AI 智能体团队产品，定位为创业公司的**运营自动化层**。与 Claude（思考与研究）和 Claude Code（产品构建）互补，Claude Cowork 专注于**让重复性运营工作在创始人不在场的情况下自动执行**。

## 产品定位

| 产品 | 核心职责 |
|------|----------|
| Claude | 研究、战略、决策支持 |
| Claude Code | 产品构建、代码生成、安全扫描 |
| **Claude Cowork** | 运营自动化、流程执行、日常任务代理 |

## 典型应用场景

### Launch 阶段
- 安排 Sprint 仪式（sprint ceremonies scheduling）
- 路由传入的 Bug 报告到正确位置
- 汇编来自连接数据源的周指标报告
- 维护用户信号流入产品决策的反馈循环

### Scale 阶段
- 企业支持工单路由与升级工作流
- 文档更新（产品变化触发）
- 续约追踪
- 分析师简报后勤
- 内容 pipeline、外发序列、CRM 卫生、pipeline 报告
- GTM 战略的战术执行层

## 核心价值主张

> "Claude Cowork then runs the operational layer of enterprise support itself: ticket routing, escalation workflows, documentation updates triggered by product changes, renewal tracking, and the reporting cadences that enterprise customer success relies on."

**让小团队具备大型组织的支持姿态**——这正是签署多年期企业合同所需要展示的能力。

## 与 Skills 体系的关联

Claude Cowork 使用 Skills 和插件系统来扩展其影响力。Airtree 案例：一人在 Cowork 内构建工作流自动化 skill，整个组织的人都可以用它完成待办事项。
