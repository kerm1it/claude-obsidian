## 概述

由 [[people/AndrewWilkinson]] 创立的加拿大企业集团，以收购和运营现金流稳定的互联网/SaaS 业务为主要模式。也持有 VC 式少数股权投资（Folly Partners 追踪的 132 笔直接投资之一部分来自 Tiny 关联结构）。

---

## AI 化进程

- 旗下软件业务面临最大 AI 冲击（定价压力、竞争激化）
- Andrew 本人 + CFO + 总裁均深度使用 Claude Code
- CFO 两周内用 Claude Code 替换了 Addepar（$5-10 万/年的投资组合管理软件）
- 使用 Claude API 管理家族办公室全部数据查询，月账单约 $4 万
