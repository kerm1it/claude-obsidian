# 自主商业运营（AI Agent 运营完整业务）

**来源：** [[people/AndrewWilkinson]]（[[entities/Tiny]] 创始人），Greg Isenberg 访谈  
**参见：** [[sources/andrew-wilkinson-ai-agents-2026-05-12]]

---

## 核心主张

> "我们大概还有三到六个月就能把业务完全交给 AI 来运营，至少是像这样的基础业务。"

AI Agent 正在接管传统企业职能，实现部分甚至完整的业务自主运营。

---

## 实战案例：Deep Personality

Andrew 用 AI Agent 完整运营了一个 SaaS 业务（Deep Personality 个性化测试产品），使用 **Harbor** 编排三类 Agent：

### Agent 分工

**Support Agent**
- 接收用户邮件工单
- 自动处理一般问题
- P0 安全问题 → 直接修复并合并 PR
- 修复完成后自动回复用户

**Dev Agent**
- 接收 Support 转来的技术问题
- 自动诊断、修复、提 PR

**Marketing Agent**
- 连接 PostHog（用户数据）+ Meta/Reddit 广告账户
- 多变量广告创意测试
- 动态设置预算
- 接受自然语言指令（"把预算提高 $1000"）

> "Support 很快就不再是一份工作。这太震撼了，真的是太震撼了。"

## 实战案例：Family Office

Andrew 的 CFO（从未写过代码）两周内用 Claude Code 替换了 **Addepar**（$5-10 万/年的投资组合管理软件），构建了：
- 全局资产负债表（实时）
- 132 笔直接投资全追踪
- 自然语言查询（压力测试、风险分析、异常检测）
- 整合 Tiny 旗下 24 家公司全部数据

家族办公室月均 Claude 账单：**$4 万**（部分替代了薪资）

## 当前局限

> "感觉像 Zapier，但能做基本决策、有智能——但你还是要手把手告诉它每一步。"

现阶段的 AI Agent：
- 还不是真正自主的 CEO/决策者
- 仍需要详细的 SOP 和 prompt
- 更像"天才婴儿"——聪明但每件事都要手把手教
- 运营者通常 50% 时间 debug、30% 时间改进，只有 20% 时间真正高效

---

## 成熟路径预测

| 时间 | 状态 |
|---|---|
| 现在（2026-04） | Support 可自主；营销可执行；CFO 可替换昂贵 SaaS |
| 3-6 个月后 | 基础 SaaS 业务可完整交给 AI 运营 |
| 1-2 年后 | 上下文窗口 5-10M → 可运营更复杂业务；"CEO 模型"出现 |
