# 自监督 WiFi 学习（对比 CSI 嵌入模型）

## 核心思想

WiFi 信号感知最大的难题之一是**标注数据稀缺**——在新房间里收集带姿态标签的 CSI 数据既繁琐又昂贵。RuView ADR-024 引入**对比自监督学习**：模型仅靠观察"哪些 CSI 片段相似、哪些不同"自我训练，无需任何人工标注，适应新环境只需 30 秒。

## 架构

```
WiFi CSI [56 子载波] → Transformer + 图神经网络（双头）
                              ├─ 128 维环境指纹（检索 + 识别）
                              └─ 17 关键点身体姿态（人体追踪）
```

**模型大小**（可运行于 ESP32）：
| 组件 | 参数量 | ESP32 内存 |
|------|--------|------------|
| Transformer 骨干 | ~28,000 | 28 KB |
| 嵌入投影头 | ~25,000 | 25 KB |
| 每房间 MicroLoRA 适配器 | ~1,800 | 2 KB |
| **合计** | **~55,000** | **~55 KB**（可用 520 KB）|

## 训练模式

| 模式 | 所需数据 | 产出 |
|------|----------|------|
| 自监督预训练 | 仅原始 CSI（无标签）| 理解 WiFi 信号结构的通用表征 |
| 监督微调 | CSI + 姿态标签 | 完整姿态追踪 + 环境指纹 |
| 跨模态 | CSI + 摄像头画面 | 与视觉对齐的指纹（ADR-079，待完成）|

## 关键技术

| 技术 | 作用 |
|------|------|
| 对比学习 | 模型自学"相似 vs 不同"，无需标签 |
| Hard-Negative Mining | 聚焦最难区分的样本对，提升样本效率 |
| MicroLoRA 适配器 | 1,792 参数/房间，适配新环境所需数据量降低 93% |
| EWC++ 正则化 | 学习新任务时不遗忘预训练知识 |
| 环境指纹索引 | 4 类索引：房间指纹 / 活动边界 / 时序基线 / 个人运动签名 |

## 环境指纹应用

128 维嵌入向量存入 [[entities/CognitumSeed]] 向量存储，支持：
- **房间识别**："这是厨房还是卧室？"
- **异常检测**："这个房间出现了意外扰动"
- **人员重识别**：跨会话追踪同一个人（WiFi 信号个人签名）
- **活动分类**：烹饪、睡眠、锻炼的自动识别

## 训练速度

- 无摄像头训练（M4 Pro）：**84 秒**
- 新房间自适应（MicroLoRA）：**30 秒以内**

## 与经典方案的关系

类似于计算机视觉中的 SimCLR / MoCo 对比学习，但应用于 CSI 时序信号而非图像 patch。将视觉领域的自监督方法迁移到 RF 感知领域。
