# 四层记忆整合

agentmemory（rohitg00）的记忆分层架构，类比人脑记忆的层级处理模型。

## 四个层级

| 层级 | 内容 | 人脑类比 | 生命周期 |
|------|------|---------|---------|
| **Working** | 工具调用的原始观察（PostToolUse hook 捕获） | 短期记忆 | 会话内 |
| **Episodic** | 压缩后的会话摘要（Stop/SessionEnd hook 触发） | "发生了什么" | 数天~数周 |
| **Semantic** | 从 Episodic 提取的事实和模式 | "我知道什么" | 持久 |
| **Procedural** | 工作流和决策模式 | "怎么做" | 持久 |

## 记忆衰减机制

基于 Ebbinghaus 遗忘曲线：
- 频繁访问的记忆自动强化
- 长期未访问的记忆自动衰减
- 矛盾的记忆自动检测并解决
- 到期记忆自动驱逐（TTL）

## 与其他 Agent 记忆系统的收敛

| 系统 | 同构模式 | 实现 |
|---|---|---|
| **agentmemory**（rohitg00） | 4 层 Working/Episodic/Semantic/Procedural | 本概念页 |
| **[[concepts/AnthropicDreaming]]**（Anthropic 官方） | 离线批处理，跨 transcript 提取模式，生成 memory diff | 独立实现 |
| **[[concepts/DreamCycle]]**（GBrain） | 夜间自动维护，Compiled Truth 重整 | 独立实现 |
| **[[concepts/AgentMemoryAPI]]**（Anthropic） | 文件系统式记忆，Agent 自主用 Bash/Grep 管理 | 不同层级，互补 |

三个独立团队（Anthropic / GarryTan / rohitg00）收敛于同一模式：**异步离线记忆巩固是 Agent 系统的通用过程层原语**。
