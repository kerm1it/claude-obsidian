# 软件民主化（印刷机类比）

**来源：** [[people/BorisCherny]]（Claude Code 创始人，Anthropic）  
**参见：** [[sources/boris-cherny-coding-solved-2026-05-12]]

---

## 核心类比

> "1400 年代，欧洲只有 10% 的人口识字……印刷机发明后 50 年，出版量超过此前一千年，书本价格下跌 100 倍……几百年后，全球识字率达到 70%。"

Boris Cherny 认为，**编程将像读写一样成为每个人都能掌握的技能**。AI 对编程的影响，与印刷机对识字率的影响在结构上是同构的。

---

## 类比结构

| 维度 | 印刷机（1400s） | AI 编程（2020s+） |
|---|---|---|
| 前状态 | 10% 人口识字，识字者为贵族/教士服务 | 少数专业工程师，服务于企业 |
| 触发事件 | 印刷机发明 | LLM + Claude Code 等工具 |
| 成本变化 | 书本价格 ↓ 100x | 编写软件成本 ↓ 10-100x |
| 普及时间 | 几百年 | 将快得多 |
| 最终状态 | 70%+ 全球识字率 | 编程成为大众基础技能 |
| 专业者仍存在 | 职业作家仍然存在 | 专业工程师仍然存在 |

---

## 推论

**领域专家 > 通用工程师**：当编程变容易时，最懂会计的人才是写会计软件的最佳人选——领域知识才是稀缺的。

**编程将成为"基础技能"**，就像今天发短信一样，不需要专业学位。

**已有信号（来自 Anthropic 内部）**：Claude Code 团队的 PM、设计师、数据科学家、财务、用户研究人员——所有人都在写代码。
