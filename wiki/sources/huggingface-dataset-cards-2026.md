# Hugging Face dataset cards

**来源**：Hugging Face Docs
**URL**：https://huggingface.co/docs/hub/datasets-cards

## 摘要

Hugging Face dataset cards 使用数据集仓库中的 README.md 展示数据集元数据、license、language、size、bias、intended use 和其他使用背景，帮助用户判断是否能负责任地使用数据。

## 关键贡献

- 将 dataset metadata 和 documentation 变成公开复用数据的默认入口。
- 强调偏差、用途和上下文信息对数据复用的重要性。
- 对本 vault 的意义：RAG、训练和 eval 数据来源需要 dataset card / datasheet 风格的文档。
