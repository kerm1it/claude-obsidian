# Model Cards for Model Reporting

**来源**：arXiv / FAT* 2019
**URL**：https://arxiv.org/abs/1810.03993

## 摘要

Model Cards 提出一种短文档框架，用于描述模型 intended use、性能特征、评估流程、跨群体和交叉群体表现、限制和相关背景。它是模型发布透明度和评估结果沟通的基础模式。

## 关键贡献

- 把性能指标放进 intended use、评估条件和适用范围中解释。
- 要求报告不同条件和群体下的 benchmarked evaluation，而不是只报告总体分数。
- 对本 vault 的意义：release card 是 model card 思路在具体变更决策上的轻量化应用。
