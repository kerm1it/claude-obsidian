## 学习路线建议

按当前 [[learning/AI学习计划-2026]]，优先顺序建议为：

1. **基础使用与限制**：Claude 101、AI Capabilities and Limitations、AI Fluency: Framework & Foundations。
2. **开发者主线**：Claude Code 101、Claude Code in Action、Claude Platform 101、Building with the Claude API。
3. **上下文与 Agent 主线**：Introduction to Model Context Protocol、Model Context Protocol: Advanced Topics、Introduction to agent skills、Introduction to subagents。
4. **产品/组织/行业应用**：Claude Cowork、AI Fluency 系列、Bedrock、Vertex AI。

## 回填到 AI 知识体系

- 主归属：学习资源，不是新增 AI 顶层。
- 覆盖层级：3 推理与能力层、4 上下文与知识层、5 Agent 系统层、6 评估与可靠性层、7 产品与组织层。
- 对当前学习最有价值的课程：Claude Code 101、Claude Code in Action、Introduction to Model Context Protocol、Introduction to agent skills、Introduction to subagents。
- 完成课程后应回填：[[learning/AI学习计划-2026]] 进度、相关概念页、必要时新增课程学习产出笔记。
