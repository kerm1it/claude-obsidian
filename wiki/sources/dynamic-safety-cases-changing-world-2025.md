# Safe Autonomous Systems in a Changing World

**来源**：Safety Science / ScienceDirect
**URL**：https://www.sciencedirect.com/science/article/pii/S0925753525001900

## 摘要

论文讨论 dynamic safety cases，指出开放环境中的自主系统不能只依赖设计时 safety case。系统运行期间的新知识、runtime data、上下文变化和不确定性，都需要进入持续 assurance。

## 关键贡献

- 把 static safety case 的局限明确连接到 deployment validity 和 runtime robustness。
- 强调 dynamic safety assurance 需要 monitorability、assessability、updateability。
- 主张在自动化更新和 human oversight 之间取得平衡。
- 对本 vault 的意义：AI release evidence 需要 runtime freshness 状态，而不是一次性审批后永久有效。
