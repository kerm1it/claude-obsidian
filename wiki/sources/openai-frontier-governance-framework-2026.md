# OpenAI Frontier Governance Framework

**来源**：OpenAI
**URL**：https://openai.com/index/openai-frontier-governance-framework/

## 摘要

OpenAI Frontier Governance Framework 说明其安全和安全保障实践如何与新兴法规对齐，覆盖风险评估、缓解、模型报告、安全风险管理、incident response、外部专家输入和框架更新。

## 关键贡献

- 将 Preparedness Framework 作为严重风险管理的基础。
- 覆盖 cyber、CBRN、harmful manipulation、loss of control 等 frontier risk。
- 强调模型报告、安全风险管理、incident response、外部专家输入和框架更新。
- 对本 vault 的意义：高风险监控升级链不止到工程 on-call，还要能进入 safety governance、external input 和 leadership/board oversight。
