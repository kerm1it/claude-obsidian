# Model Release Notes

**来源**：OpenAI Help Center
**URL**：https://help.openai.com/en/articles/9624314-model-release-notes

## 摘要

OpenAI Model Release Notes 记录 ChatGPT 和 API 模型更新、retirement、sunset period、fallback 和功能变化。页面会随产品更新持续变化。

## 关键贡献

- 模型更新会改变响应风格、质量、功能入口、fallback 和可选模型集合。
- 模型 retirement / sunset 需要提前通知、迁移和影响面检查。
- release notes 是 release gate 的外部沟通与 audit 输入。
- 对本 vault 的意义：AI 系统依赖 provider 时，需要把 release notes 变成监控输入，而不是等用户发现行为变化。

- 交叉链接：3. 推理与能力层、6. 评估与可靠性层
