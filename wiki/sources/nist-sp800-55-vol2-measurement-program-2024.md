# NIST SP 800-55 Vol. 2 Developing an Information Security Measurement Program

**来源**：NIST SP 800-55 Vol. 2
**URL**：https://csrc.nist.gov/pubs/sp/800/55/v2/final

## 摘要

NIST SP 800-55 Vol. 2 提供如何建立 information security measurement program 的指南，强调 measurement 不只是选择几个指标，而是组织化地实施、沟通、维护和改进 measures。

## 关键贡献

- Measurement program 需要灵活结构，覆盖 measure development 和 implementation 活动。
- 指标需要进入组织沟通、责任和持续改进流程。
- 与 Vol. 1 的 measure selection 配套，Vol. 2 更关注 program governance。
- 对本 vault 的意义：governance dashboard 需要 owner、cadence、review、communication path 和 retire/update 机制。
