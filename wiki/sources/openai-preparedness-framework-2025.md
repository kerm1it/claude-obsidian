# OpenAI updated Preparedness Framework

**来源**：OpenAI
**URL**：https://openai.com/index/updating-our-preparedness-framework/

## 摘要

OpenAI updated Preparedness Framework 是 OpenAI 追踪、评估和准备 frontier model severe-risk capability 的流程。它包含 capability reports、safeguards reports、Safety Advisory Group review 和领导层部署决策。

## 关键贡献

- 把模型能力风险评估接到 safeguards 和部署决策。
- Safety Advisory Group 提供跨职能 review 和建议。
- High / Critical capability thresholds 将能力证据与 required safeguards、部署限制和 leadership decision 绑定。
- 对本 vault 的意义：AI safety ops 的核心是从 eval evidence 到 release gate 的组织链路。
- 对 override governance 的意义：frontier deployment 的 exception 必须接到 safety review、stronger safeguards、further evaluation 和有权责任人，而不是产品单方绕过。
- 对 override debt dashboard 的意义：单次 residual risk review 还需要跨时间趋势，才能发现例外是否在侵蚀部署标准。

## 对 AI 知识体系的归属

- 主归属：7. 产品与组织层
- 交叉链接：6. 评估与可靠性层、8. 自我改进与前沿层
