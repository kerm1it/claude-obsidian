# What Keeps Agent Skills from Being Reusable?

**来源**：OpenReview / Agent Skills 2026 Workshop
**URL**：https://openreview.net/attachment?id=n0AIlfxDU0&name=pdf

## 摘要

这项研究分析大量公开 SKILL.md 文件，关注 agent skills 为什么难以复用。搜索摘要显示研究样本约 138K 个公开 SKILL.md 文件，并指出 routing defect、环境依赖和格式/metadata 问题会影响可复用性。

## 关键贡献

- Skill 可复用性不仅取决于是否符合格式。
- Metadata、routing signal、环境要求、依赖和说明质量会影响 agent 是否能正确调用。
- 大型 skill library 需要质量控制、结构化索引和退役机制。
