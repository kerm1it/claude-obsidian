# RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control

**来源**：arXiv:2307.15818
**URL**：https://arxiv.org/abs/2307.15818

## 摘要

RT-2 研究如何把 web-scale vision-language pretraining 直接接入机器人控制。论文把机器人动作表达为文本 token，并把视觉、语言和动作放入同一训练格式，形成 vision-language-action model。

## 关键贡献

- 多模态能力进入真实世界行动时，需要 action modality。
- RT-2 同时训练机器人轨迹数据和互联网视觉语言任务。
- 模型在新对象、未见指令和简单语义推理上有更强泛化。
- VLA 是第 3 层能力进入第 5 层 embodied agent 的关键桥梁。

## 对 AI 知识体系的归属

- 主归属：5. Agent 系统层
- 交叉链接：3. 推理与能力层、6. 评估与可靠性层
- 作用：说明多模态世界理解必须通过 action/control 才能变成机器人能力。
