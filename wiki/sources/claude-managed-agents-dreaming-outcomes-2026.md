# New in Claude Managed Agents: dreaming, outcomes, and multiagent orchestration

**来源**：Claude Blog
**URL**：https://claude.com/blog/new-in-claude-managed-agents

## 摘要

Anthropic 将 dreaming、outcomes 和 multiagent orchestration 放在同一次 Managed Agents 更新中发布，说明生产级自我改进依赖三件事：可持续记忆、可评分结果、可编排执行。

## 关键贡献

- Dreaming 负责在会话之间整理 memory。
- Outcomes 负责用 rubric 或目标对 agent 输出评分。
- Multiagent orchestration 负责并行和委派复杂任务。

## 对 AI 知识体系的归属

- 主归属：8. 自我改进与前沿层
- 交叉链接：5. Agent 系统层、6. 评估与可靠性层
- 作用：把 dream 放入 memory + outcome + orchestration 的完整系统，而不是单独功能。
