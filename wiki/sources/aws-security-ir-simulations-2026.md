# AWS Security Incident Response Guide: Types of simulations

**来源**：AWS Documentation
**URL**：https://docs.aws.amazon.com/security-ir/latest/userguide/types-of-simulations.html

## 摘要

AWS Security Incident Response Guide 区分 tabletop exercise、functional exercise 和 full-scale simulation 等演练形式，用来逐步验证 incident response 计划和团队协作。

## 关键贡献

- Tabletop 更适合 discussion-based plan validation。
- Functional exercise 可验证具体流程、工具和角色执行能力。
- Full-scale simulation 更接近真实响应，但需要更高控制和保护。
- 对本 vault 的意义：AI runbook drill 可以从 tabletop 开始，逐步升级到受控技术演练，例如 provider fallback 或 rollback drill。
