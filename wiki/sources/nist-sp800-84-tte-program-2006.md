# NIST SP 800-84: Guide to Test, Training, and Exercise Programs

**来源**：NIST SP 800-84
**URL**：https://csrc.nist.gov/pubs/sp/800/84/final

## 摘要

NIST SP 800-84 是 IT plan/capability 的 test、training、exercise 程序指南。它给 tabletop exercise、functional exercise 和 exercise evaluation 提供结构化流程。

## 关键贡献

- 演练需要先定义目标、范围、参与者、场景和评价方式。
- Tabletop exercise 适合通过讨论验证计划、角色、沟通和决策。
- 演练后必须形成 findings、lessons learned 和 improvement plan。
- 对本 vault 的意义：monitor tabletop / runbook drill 应被看作组织能力测试，有明确评价和改进行动。
