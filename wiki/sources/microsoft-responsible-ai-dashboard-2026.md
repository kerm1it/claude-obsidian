# Microsoft Responsible AI Dashboard

**来源**：Microsoft Learn
**URL**：https://learn.microsoft.com/en-us/azure/machine-learning/concept-responsible-ai-dashboard

## 摘要

Microsoft Responsible AI Dashboard 将 model overview、error analysis、fairness assessment、data analysis、interpretability、counterfactual analysis 和 causal inference 组合成可定制的模型调试与决策工作流。

## 关键贡献

- Dashboard 支持 cohort / disaggregated analysis，避免一个整体指标掩盖 blind spots。
- 组件之间共享 dataset cohorts，让 error、fairness、data distribution 和 interpretability 能被并排分析。
- Dashboard 输出可分享的 scorecard，支持产品和业务 stakeholder 审查。
- 对本 vault 的意义：AI governance dashboard 需要 slice、cohort、scorecard 和可解释证据，而不是单个总分。
