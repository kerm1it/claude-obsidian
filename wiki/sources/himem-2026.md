# HiMem: Hierarchical Long-Term Memory for LLM Long-Horizon Agents

**来源**：arXiv:2601.06377
**发布时间**：2026-01-10

## 摘要

HiMem 认为长期 memory 系统在持续交互中仍存在适应性、可扩展性和自演化限制，因此提出层级长期记忆框架，支持 memory construction、retrieval 和 dynamic updating。

## 关键贡献

- Memory 是长期交互 agent 的结构性问题，不只是检索问题。
- 层级结构用于支持长周期对话中的构建、检索和动态更新。
- 与本 vault 的 [[concepts/MemoryTree]] 和 [[concepts/四层记忆整合]] 同方向。
