# FEMA Homeland Security Exercise and Evaluation Program

**来源**：FEMA HSEEP
**URL**：https://www.fema.gov/emergency-managers/national-preparedness/exercises/hseep

## 摘要

FEMA HSEEP 提供 exercise design、conduct、evaluation、after-action report 和 improvement planning 的通用方法。虽然面向 broader emergency preparedness，但 evaluation/improvement loop 对 AI incident drill 同样适用。

## 关键贡献

- 演练需要从目标和能力出发设计，而不是只模拟事件。
- Evaluation 要收集观察证据，形成 after-action report。
- Improvement plan 要有责任人、截止日期和可验证关闭。
- 对本 vault 的意义：AI runbook drill 只有进入 AAR/IP，并回填 monitor、eval、release gate 和 incident playbook，才算形成闭环。

## 对 AI 知识体系的归属

- 主归属：7. 产品与组织层
- 交叉链接：6. 评估与可靠性层
