# The next evolution of the Agents SDK

**来源**：OpenAI
**URL**：https://openai.com/index/the-next-evolution-of-the-agents-sdk/
**发布时间**：2026-04-15

## 摘要

OpenAI 介绍 Agents SDK 的新能力：面向 OpenAI 模型构建的标准化 agent 基础设施，包括 model-native harness、跨文件和工具的电脑操作，以及安全的 native sandbox execution。

## 关键贡献

- “harness”不只是 prompt 模板，而是 agent 运行时：受控工作区、明确指令、工具、文件、沙盒和执行记录。
- 沙盒执行把 agent 行动限制在可控环境内，是可靠性和安全性的基础。
- 这说明 agent 工程正在从“会调用工具”走向“可观测、可评估、可控制的运行时系统”。

## 对 AI 知识体系的归属

- 主归属：5. Agent 系统层
- 交叉链接：6. 评估与可靠性层
- 下游：产品化 agent、coding agent、research agent、computer-use agent。
