# Review annotations in Label Studio

**来源**：HumanSignal / Label Studio Docs
**URL**：https://docs.humansignal.com/guide/quality.html

## 摘要

Label Studio 的 annotation review 工作流强调复审标注、查看 annotator performance、consensus 和项目质量状态，目标是避免低质量标注进入模型训练数据。

## 关键贡献

- 把 annotation review 固定为数据生产流程的一环，而不是事后抽查。
- 提供 annotator performance、consensus、review workflow 等平台级质量原语。
- 对本 vault 的意义：human feedback ops 需要可执行的 review / QA workflow，不只是抽象的“请人标注”。
