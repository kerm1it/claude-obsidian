# Quality analysis

**来源**：Labelbox Docs
**URL**：https://docs.labelbox.com/docs/quality-analysis

## 摘要

Labelbox quality analysis 文档将 label quality 分成 benchmark / gold standard 和 consensus 等质量分析方式，用于评估标注准确性和标注员表现。

## 关键贡献

- 固定 gold standard / benchmark task 和 consensus as QA primitives。
- 展示标注平台如何把质量分析嵌入数据流水线。
- 对本 vault 的意义：annotation ops 的稳定 QA 单元是 gold、overlap、consensus、review 和 rework，而不是只看标注吞吐。
