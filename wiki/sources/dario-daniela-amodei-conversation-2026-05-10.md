## 内容概览

Anthropic 联合创始人 Dario 和 Daniela Amodei 在 Code with Claude 开发者大会上接受 CPO Ami Vora 采访，谈及 Anthropic 的增长现状、对开发者生态的看法、产品哲学、模型能力趋势以及公司使命。

---

## 核心要点

### 指数级增长现实

- Anthropic 内部有一个"过山车"Slack emoji，象征这段旅程
- 预期：每年 10x 增长；实际 Q1 2026：年化 **80x** 增长（营收 + 用量）
- 超出预期导致计算资源压力，故有 **SpaceX 算力合作**公告
- Dario 坦言：希望 80x 不要持续，"10x 已经足够疯狂了"

### 规模律与预测准确性

- Dario 和联合创始人在 10 年前通过 scaling laws 预测了今天的模型能力曲线
- 预测写在图表上，每一步都准确兑现，但"亲眼目睹仍然令人震惊"
- 类比：《星际穿越》中的黑洞引力波——数学上都知道，但亲见时仍觉不可思议

### 开发者是最重要的用户

- Daniela：开发者是最重要的用户群，因为他们给真实、诚实的反馈
- Anthropic 从创立起就以开发者和企业为主要服务对象
- Dario：软件工程师是最快采纳新技术的群体，是 AI 扩散到全经济的先导

### 单人十亿美金公司

- 约一年前 Dario 预测：**2026 年将出现首个"单人十亿美金公司"**
- 现状：已有双人团队市值 10 亿、单人创始人价值数亿美金
- 剩余 7-8 个月，仍有可能实现 → 详见 [[concepts/单人十亿美金公司]]

### 多 Agent 与"数据中心里的天才国家"

- 趋势一：从单 Agent → 多 Agent 协同，"管理一整个 quad 团队"
- 趋势二：从个人生产力 → 整个团队/组织的生产力
- 趋势三：Amdahl 定律——代码写快了，但安全、验证、代码质量没跟上
- 终极愿景：**"数据中心里的天才国家"**（country of geniuses in a data center）
- 详见 [[concepts/国家级天才数据中心]]

### 产品哲学：为指数构建

- 新能力在指数的某个台阶"突然点亮"——此前 Claude Code 在 2022 年尝试但失败（模型太弱）
- 产品会达到"饱和"：chatbot 已触顶，Agent 形态仍在上升
- API 永远不会消失，因为模型变强 → 新产品可能 → 持续创新
- "将 AI 加速用于 AI 自身加速"：Claude 帮助训练 Claude，Claude Code 帮助写 Claude
- 风险：AI 写得越快，技术债积累越快 → 需要 AI 来偿还技术债

### Anthropic 使命与 Hold Light and Shade

- 两大支柱：①负责任地开发变革性技术；②确保技术对所有人有益
- 核心文化价值：**Hold Light and Shade**（同时持有光明与阴影）
- Opus 和 Glass 发布谨慎，因安全漏洞考虑 → 典型的"光影同持"
- 详见 [[concepts/HoldLightAndShade]]

### 开发者使用案例

Daniela 分享最感动的案例：
- 全球南方移动医生接口（帮助偏远地区获得医疗建议）
- AI 加速生物医学研究
- 用 Claude 从损坏硬盘恢复婚礼照片
- 用 Claude 追踪花园番茄生长曲线
